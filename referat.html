jQuery е бърза библиотека на JavaScript с много функционалности. Тя прави управлението на HTML документи, обработката на събития, анимирането и AJAX много по-лесно, използвайки програмен интерфейс, който работи в по-голямата част от съвременните браузъри. Със своята гъвкавост и разшируемост, jQuery променя начина на писане на JavaScript на милиони хора, улеснявайки тяхната работа.

Обработка на събития

1 | var hiddenBox = $( "#banner" );
2 | $( "#button-container button" ).on( "click", function( event ) {
3 | 	hiddenBox.show();
4 | });

Ajax

Извикване на локален скрипт на сървъра /api/getWeather с query параметър zipcode=2300 и смяна на HTML кода на елемента #weather-temp с върнатия текст. 

1 | $.ajax({
2 |		url: "/api/getWeather",
3 |		data: {
4 |			zipcode: 2300
5 |		},
6 |		success: function( result ) {
7 |			$( "#weather-temp" ).html( "<strong>" + result + "</strong> degrees" );
8 |		}
9 | });
